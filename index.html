Boulder Farm Map

Interactive story map and tour of farms in Boulder, CO using React and SVG

Boulder Farm Map

Live Site | GitHub Frontend | GitHub Backend

This project is a listing of farms and an integrated svg map that is based on a project by the Boulder Convention and Visitors Bureau called Farm Trail. Farm Trail is a paper brochure that features a map of farms in Boulder county and a listing of the farms with the various activities that are available at each farm such as farm dinners, workshops, volunteer opportunities, etc.

I really wanted to have some sort of mapping element in this project so I contacted the Boulder CVB to see if I could make an interactive version of their Farm Trail brochure and they said yes.

The main feature of this site is a scrollable listing of 21 area farms with a photo, location, description and contact info.

At the bottom of most listings is a series of colored icons representing the different activities each farm offers to visitors.

You can hover over these icons and see a popup of what they represent such as tours, workshops, farm dinners, and volunteer opportunities.

These farm listings are connected to the map on the right and as you scroll through the farms, you can see the corresponding map marker highlighted.

We can also click on each marker to trigger a scroll to the corresponding farm.

The map is an SVG that I created in Adobe illustrator and the frontend is built with React. To integrate the map with the farm listings I took the code from the svg file and placed it inside a react component so it could be manipulated to interact with other components and receive state.

The Farm listings are populated by a Rest API I created for the backend using Node.js, Express, and Knex.
